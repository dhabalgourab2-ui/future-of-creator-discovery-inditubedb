# What Is IndiTubeDb?

IndiTubeDb is an AI-powered creator discovery platform that helps users discover, compare, evaluate, and learn from YouTube creators using community intelligence rather than subscriber counts alone.

Think of it as:

> IMDb for YouTube Creators

or

> Rotten Tomatoes for the Creator Economy

Instead of relying on algorithms, popularity, or viral content, IndiTubeDb helps users discover creators through quality signals.
